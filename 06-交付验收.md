# 交付验收

## MVP 上线检查单

### 功能验收

- [ ] 首页正常访问，内容完整
- [ ] 无人机频率查询工具可用（筛选、搜索、结果展示）
- [ ] 工具数据覆盖 ≥ 20 条记录
- [ ] 3 篇博客文章已发布且可访问
- [ ] 博客列表页正常显示
- [ ] About 页面存在
- [ ] 所有内链正常跳转，无 404

### SEO 验收

- [ ] 每个页面有独立 title（< 60字符）
- [ ] 每个页面有 meta description（< 160字符）
- [ ] sitemap.xml 可访问（/sitemap.xml）
- [ ] robots.txt 配置正确
- [ ] Google Search Console 已提交 sitemap
- [ ] 工具页和文章页有 JSON-LD schema

### 性能验收

- [ ] 首页 LCP < 2.5s（PageSpeed Insights 测试）
- [ ] 移动端显示正常（Chrome DevTools 模拟）
- [ ] 图片已压缩优化

### 分析验收

- [ ] Google Analytics 4 数据正常上报
- [ ] Google Search Console 已验证域名所有权

### 部署验收

- [ ] 域名 counteruavhub.com 正常解析
- [ ] HTTPS 证书有效
- [ ] Git push 后 Cloudflare Pages 自动部署正常
